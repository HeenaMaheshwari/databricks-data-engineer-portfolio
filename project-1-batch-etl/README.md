# Databricks Data Engineering Portfolio

This repository demonstrates production-style data engineering workflows built on Databricks using Delta Lake and Spark SQL.

## Project Overview
The project follows the Medallion Architecture:
- **Bronze Layer**: Raw data ingestion
- **Silver Layer**: Cleaned and transformed data
- **Gold Layer**: Analytics-ready tables

## Key Features
- Databricks ETL pipelines
- Delta Lake tables
- SQL-based transformations
- Cost-efficient job cluster usage
- Production-oriented folder structure

## Technologies Used
- Databricks
- Apache Spark
- Delta Lake
- SQL

## Use Case
Designed to showcase scalable, maintainable ETL pipelines suitable for real-world analytics workloads.
